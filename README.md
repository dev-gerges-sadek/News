# 📰 News App 2

A modern Flutter-based news application that fetches and displays the latest news articles in a clean and user-friendly interface.

---

## 📌 Description

**News App 2** is a simple yet powerful mobile application built with Flutter. It allows users to browse news articles, open full stories in the browser, and stay updated with current events.

---

## 🚀 Features

* 🗞️ Fetch latest news from API
* 📱 Clean and responsive UI
* 🔗 Open full articles in browser
* ⚡ Fast networking with Dio
* 🌐 External link handling
* 📂 Asset support for images

---

## 🛠️ Tech Stack

* **Flutter SDK**
* **Dart**
* **Dio** (HTTP client)
* **URL Launcher** (open links in browser)

---

## ▶️ Getting Started

1. Clone the repository:

```bash
git clone https://github.com/your-username/news_app2.git
```

2. Navigate to project folder:

```bash
cd news_app2
```

3. Install dependencies:

```bash
flutter pub get
```

4. Run the app:

```bash
flutter run
```
